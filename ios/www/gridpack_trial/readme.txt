To get information on how to configure dhtmlxGridPack, explore our documentation: https://docs.dhtmlx.com/gridpack.
You can import dhtmlxGridPack into your projects using yarn or npm package manager. To get the trial version of dhtmlxGridPack, run the following commands:

// yarn
yarn config set @dhx:registry https://npm.dhtmlx.com
yarn add @dhx/trial-gridpack

Or

// npm
npm config set @dhx:registry https://npm.dhtmlx.com
npm i @dhx/trial-gridpack
